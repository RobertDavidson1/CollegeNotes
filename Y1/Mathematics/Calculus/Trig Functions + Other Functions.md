## Trigonometry (x is in radians)
| Function | Derivative |
|:-------- |:----------:|
| sin(x)   |  $cos(x)$  |
| cos(x)   | $-sin(x)$  |
| tan(x)   | $sec^2(x)$ |
___
## Inverse Trigonometry 
| Function      |        Derivative         |
|:------------- |:-------------------------:|
| $sin^{-1}(x)$ | $\dfrac{1}{\sqrt{1-x^2}}$ |
| $cos^{-1}(x)$ | $\dfrac{1}{\sqrt{1-x^2}}$ |
| $tan^{-1}(x)$ |    $\dfrac{1}{1+x^2}$     |
___
## Common Functions
| Function     | Derivative                   |
| ------------ | ---------------------------- |
| $\sqrt x$    | $\dfrac{1}{2} \cdot x^{1/2}$ |
| $\large a^x$ | $ln(a)\;a^x$                 |
| $\large e^x$ | $\large e^x$                 |
| $ln(x)$      | $\dfrac{1}{x}$               |
| $log_a(x)$   | $\dfrac{1}{x\cdot ln(a)}$    |
